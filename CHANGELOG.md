# Changelog

## [0.1.0] - 2026-06-11

### Added

- YAML-based mapping for nested financial JSON responses
- Local JSON and authenticated HTTP JSON sources
- Required-field, numeric, and date validation
- Formatted Excel report with an auto-expanding table and chart
- Interactive Streamlit dashboard
- Sample data, tests, CI, and contributor documentation
